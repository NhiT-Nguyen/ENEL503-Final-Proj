# ENEL 503: Computer Vision Final Project

## Traffic Sign Recognition and Classification Using CNN ##

Using the dataset from [“German Traffic Sign Benchmarks,” benchmark.ini.rub.de.](https://benchmark.ini.rub.de/gtsrb_dataset.html),
this project seeks to implement computer vision filtering techniques to improve
CNN classifier performance.

The codebase is most easily run in Google Colab as a notebook. Scripts are included to download the dataset
to the instance, and takes approximately 15-20 minutes to unpack all 50,000+ images.
